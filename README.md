# Teensy 4.0 Breakout
Breakout Board for the Teensy 4.0. The connectors are designed to match the Pixhawk standard connectors, making it easy to interface with different sensors.

Features:
- 6 - 36 V Input
- 2 x Serial ports
- 1 x SPI with 2x CS
- 2 x I2C Ports
- 3 x PWM / ADC
